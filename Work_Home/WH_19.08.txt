1) Linux - это ядро операционной системы, дистрибутивов которой существует огромное колличество, 
преимущественно используется для работы с сервером и разработчиками для разработки.
2) CLI - комнандная строка, GUI - графический интерфейс
3) SSH - это сетевой протокол, другими словами формат ключа, который можно сгенерировать локально, 2 его версии, публичную и приватную
4) Сгенерировать ключи можно с помощью команд - 
(1. ssh-keygen -t rsa - команда запускающая генерацию ключей, 
2. Enter file in which to save the key (локальный путь, по которому будут находится ключи): - так же можно ввести не стандартный путь
3 Enter passphrase (empty for no passphrase): - система предложит ввести кодовую фразу для доп. защиты ключей (не обязательный шаг))
После введения этих команд сгенерируется ключ, далее нужно ввести команду для его копирования и позже вставки в соответствующее поле на сервере для SSH ключа.
Далее следует выполнить команду  "ssh root@[IP-адрес сервера]" , после этих действий больше не потребуется всегда вводить пароль при входе на сервер.
Это и есть главная причина использования этой технологии - удобство и надежность.
4) Смотрите инструкцию в пункте "3)".
5) Команда "gzip" позволяет быстро заархивировать выбраный файл что сократит его размер, (особенно текстовые файлы).
6) Это делается с помощью команд "|" и ">" - (если это первая запрись) или ">>" (если дозапись информации) пример: "df -h | > /tmp/fileOleg.txt", вывод команды df -h
запишется, перезапишется или дозапишется в указанный файл.
